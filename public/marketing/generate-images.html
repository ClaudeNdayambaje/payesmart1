Générateur d'images d'interface pour les tarifs

Cette page génère des images représentatives des interfaces pour chaque niveau de tarif. Vous pouvez télécharger ces images et les utiliser dans votre site marketing.

Instructions d'utilisation

  1. Cliquez sur les liens de téléchargement pour enregistrer les images générées
  2. Placez ces images dans le dossier /public/marketing/img/pricing/
  3. Les images seront automatiquement affichées dans les cartes de tarif sur votre site
